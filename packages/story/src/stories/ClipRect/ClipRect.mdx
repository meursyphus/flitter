# ClipRect

This widget clip the child widget like ClipRect Widget in Flutter.

<b>The ClipRect</b> take clipper that is callback function and return <b>Rect</b> object.

This widget use <b>ClipPath</b> internally to clip the child widget

See: [https://api.flutter.dev/flutter/widgets/ClipRect-class.html](https://api.flutter.dev/flutter/widgets/ClipRect-class.html)
