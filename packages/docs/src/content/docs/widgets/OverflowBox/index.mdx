# OverflowBox

This widget motivated by OverflowBox in Flutter.

> A widget that imposes different constraints on its child than it gets from its parent, possibly allowing the child to overflow the parent.

See: [https://api.flutter.dev/flutter/widgets/OverflowBox-class.html](https://api.flutter.dev/flutter/widgets/OverflowBox-class.html)
